## Table of Contents

- [Build](#build)

## Build

```bash
mkdir cpp
git clone https://github.com/ssilenzi/cpp-template.git cpp
cd cpp
mkdir build
cd build
cmake ..
make -j
```

## Usage

Run your executables:

```bash
chmod +x exec
./exec
```
